mapgentest_transparent_base_nodes (Transparent base/solid nodes) bool true
    Enable transparency for base (solid) nodes (mapgen_solid, block_loaded). The fallback and edge nodes are always opaque.

mapgentest_transparent_water_nodes (Transparent water nodes) bool true
    Enable transparency for water nodes (mapgen_water). The fallback and edge nodes are always opaque.

mapgentest_block_loaded_callback (block_loaded callback) bool true
    Enable the on_block_loaded callback that marks loaded mapblocks with the block_loaded node.
